## How to Use

### Option 1: Run via NPM Script (Recommended)
```bash
npm run add-master-products
```

### Option 2: Run Directly
```bash
node scripts/run-master-products.js
```

### Option 3: Run SQL File Directly
If you prefer to run the SQL file directly in your database:
```bash
psql -d your_database_name -f scripts/add-master-products.sql
```

## After Running

The script will:
1. Add 500+ products to your database
2. Show total product count
3. Display sample products added

## Features

✅ **Organized by Category**: Products are logically grouped  
✅ **Multiple Sizes/Models**: Each product type has various sizes and specifications  
✅ **Realistic Pricing**: Based on Sri Lankan market rates  
✅ **Stock Quantities**: Reasonable initial stock levels  
✅ **Measurements Included**: All products have their measurements in the name  
✅ **No Duplicates**: Uses `ON CONFLICT DO NOTHING` to prevent duplicate entries  
✅ **Performance Optimized**: Includes additional indexes for better query performance  

## Database Structure

Each product has:
- `id`: Auto-incrementing primary key
- `name`: Product name with size/model (e.g., "Hammer Claw 500g")
- `price`: Price in LKR (Sri Lankan Rupees)
- `stock`: Initial stock quantity
- `created_at`: Timestamp of creation
- `updated_at`: Timestamp of last update

## Notes

- Prices are in Sri Lankan Rupees (LKR)
- Stock quantities are initial values and can be adjusted
- Products include measurements in their names for easy identification
- The script won't duplicate products if run multiple times (uses `ON CONFLICT DO NOTHING`)

## Customization

To modify products:
1. Edit `scripts/add-master-products.sql`
2. Adjust prices, stock, or add new products
3. Run the migration script again
